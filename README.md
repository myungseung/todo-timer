# ⏱️ Daily Time Tracker

하루 동안 시간을 어떻게 썼는지 상세하게 기록하는 Habit Tracker

## 💡 핵심 개념

이 앱은 **단순한 시간 추적기**입니다:
- 타이머를 시작하면 그 순간부터 **초 단위로 시간을 기록**
- **매 초마다 자동 저장** - 새로고침해도 데이터 손실 없음 (최대 1초)
- ESC나 Cmd+Enter로 중단해도 **항상 경과 시간 기록**
- 각 작업에 실제로 투입한 시간을 정확하게 추적
- 프로그램적인 "완료" 기능은 없음 - 완료는 사용자가 체크박스를 클릭할 때만

## ✨ 주요 기능

### 시간 추적
- **50분 타이머** - 원형 파이 차트로 시각화 (초기값일 뿐)
- **초 단위 정확한 기록** - 매 초마다 localStorage에 자동 저장
- **기존 시간 반영** - 작업의 누적 시간에서 타이머 시작 (50분 초과 시 순환)
- **반시계방향 진행** - 10시 방향에서 시작하여 12시 방향(0분)으로
- **실시간 파이 차트** - 매 초마다 검은색 증가, 빨간색 감소
- **실시간 색상 변경** - 작동 중일 때 붉은색으로 표시

### 작업 관리
- **계층적 구조** - 3단계 들여쓰기 지원 (Tab 키)
- **실시간 시간 표시** - 각 작업별 누적 소요 시간 (mm:ss + 🍅pom)
- **포모도로 단위** - 50분 = 1 pom (0.1 단위로 표시)
- **총 소요 시간** - 하루 동안 기록된 총 시간 + 🍅 총 pom
- **단순 기록** - 완료는 체크박스로만, 프로그램은 시간만 추적
- **자동 저장** - localStorage를 통한 날짜별 데이터 보존

### PWA 지원
- **오프라인 작동** - Service Worker 캐싱
- **앱으로 설치** - Chrome, Safari, Edge 등 지원
- **자동 업데이트** - vite-plugin-pwa로 새 버전 자동 감지 및 적용
- **네이티브 앱 경험** - 독립 창 실행

## 🎯 사용법

### 시간 추적
- **Cmd + Enter** (Mac) / **Ctrl + Enter** (Windows) - 타이머 시작/중지
- **ESC** - 타이머 중단 (경과 시간 자동 기록)
- 타이머는 현재 포커스된 작업 항목에 연동
- 중간에 멈추든, 끝까지 완료하든 **항상 경과 시간이 기록됨**

### 작업 관리
- **Enter** - 새 작업 추가
- **Tab** - 들여쓰기 (하위 항목으로 변환)
- **Shift + Tab** - 들여쓰기 해제
- **Backspace** (빈 입력) - 작업 삭제
- **↑/↓** - 작업 간 이동
- **Checkbox** - 완료 처리

## 📊 데이터 구조

단순하고 명확한 데이터 구조:

```javascript
{
  "2024-01-01": {
    todos: [
      { id: 1, text: "작업", level: 0, completed: false, timeSpent: 3000 },
      { id: 2, text: "하위", level: 1, completed: true, timeSpent: 1800 }
    ]
  }
}
```

**핵심 규칙:**
- 날짜별로 데이터가 분리됨
- 각 todo는 `text` (제목)와 `timeSpent` (초 단위 시간)만 기록
- `completed`는 사용자가 체크박스를 클릭할 때만 업데이트됨
- 프로그램적인 "완료" 개념은 존재하지 않음 - 단순 시간 추적기

모든 데이터는 브라우저의 **localStorage**에 **매 초마다 자동 저장**됩니다.

## ⏱️ 타이머 및 파이 차트 설정

### 타이머 초기값
- **50분 (3000초)** - 기본 타이머 시작 시간
- 작업에 이미 기록된 시간이 있으면 그만큼 차감하여 시작
  - 예: 10초 기록 → 49:50부터 시작
  - 예: 3010초 (50분 10초) 기록 → 49:50부터 시작 (50분 초과분 순환)

### 파이 차트 구조
원형 차트는 **60분 (3600초)** 기준:
- **배경 영역 (검은색)**: 10시~12시 = 10분 고정
- **타이머 영역**: 12시~10시 = 50분
  - **경과 시간 (검은색)**: 12시부터 시계 방향으로 증가
  - **남은 시간 (빨간색)**: 경과 후부터 10시까지

### 시간 기록 로직
```javascript
// 매 초마다 실행
setInterval(() => {
  timerSeconds -= 1              // UI 카운트다운
  onTodoTimeUpdate(todoId, 1)    // localStorage에 +1초 저장
}, 1000)
```

**장점:**
- 새로고침해도 안전 (최대 1초만 손실)
- 중단 시 별도 계산 불필요
- 표시는 저장된 값 그대로 사용
